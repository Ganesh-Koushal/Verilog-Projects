## Counters and Timers

This folder contains counter-based designs and timeout logic commonly used in
control and timing applications.

### Modules
- `Counters.v` – Basic up-counter
- `Counters_timeout.v` – Counter with timeout detection
- `counter_timeoutpulse.v` – Counter with single-cycle done pulse generation

### Concepts Covered
- Counter design
- Timeout detection
- Pulse generation from level signals
- Parameterized counting limits

### Skills Demonstrated
- Designing reusable counter logic
- Handling terminal count conditions
- Generating clean control pulses
